    為什麼會有這篇文章?

    之前我發佈了一篇如何安裝hexo+在github上建立個人網頁，因為內容只寫到如何建立起網站，但並沒有說到如何建立文章，故寫下這篇當作補充。

    這篇文章做什麼用的?

    這篇文章是來說明如何在本地端建立起文章，並將文章上傳至github的網頁上。
    另外此篇也會寫如何在文章裡放置圖片，以及刪除文章。

    前置步驟

    1.在開始實作這篇文章的內容前，請先實作完這篇文章如何安裝hexo+在github上建立個人網頁，因為這篇屬於上述文章的延伸。
    2.請先了解Markdown語法，因為文章是用Markdown寫的。
